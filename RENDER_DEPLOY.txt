==========================================
 RENDER.COM DEPLOYMENT - QUICK GUIDE
==========================================

STEP 1: Push to Git
-------------------
git add .
git commit -m "Ready for Render deployment"
git push origin main


STEP 2: Deploy on Render
-------------------------
1. Go to: https://dashboard.render.com
2. Click: "New +" → "Blueprint"
3. Connect your repository
4. Render auto-detects render.yaml
5. Click: "Apply"


STEP 3: Connect UI to API (REQUIRED)
-------------------------------------
After both services deploy:

1. Get API URL from Dashboard → webtp-api → copy URL
   (e.g., https://webtp-api.onrender.com)

2. Dashboard → webtp-ui → Environment → Add:
   API_BASE_URL=https://webtp-api.onrender.com


STEP 4: Set API Keys (Optional)
--------------------------------
For AI features (or use mock mode):

Dashboard → webtp-api → Environment → Add:
  ANTHROPIC_API_KEY=sk-ant-...
  or
  OPENAI_API_KEY=sk-...

Dashboard → webtp-ui → Environment → Add:
  (same keys as above)

For mock mode (no API keys):
  USE_MOCK_LLM=true


STEP 5: Access Your App
------------------------
After ~5-10 minutes:
  UI:  https://webtp-ui.onrender.com
  API: https://webtp-api.onrender.com/docs
